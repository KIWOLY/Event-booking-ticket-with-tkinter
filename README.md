🎟️ Event Booking Ticket System

This project is an Event Booking Application with a Tkinter-based desktop frontend and a Django REST Framework (DRF) backend. It allows users to browse events, book tickets, and manage bookings in a simple and interactive way.

✨ Features
🖥️ Frontend (Tkinter)

User-friendly desktop interface

Event browsing and ticket booking forms

Booking confirmation display

Simple navigation between different windows

⚙️ Backend (Django REST Framework)

RESTful API for events and bookings

User registration and authentication

Event management (CRUD operations)

Ticket booking and tracking

Secure database storage

💡 Purpose

The system is designed to make event ticket booking faster and more accessible by combining a lightweight desktop interface with a powerful backend API.

🛠️ Tech Stack

Frontend: Python Tkinter

Backend: Django REST Framework (DRF)

Database: SQLite3

API Communication: HTTP between Tkinter and DRF
